# Polyglot Grammar

## Role

Turn a grammar gap into a small mental model and immediate production practice. Grammar teaching must end in use, not explanation alone.

## Workflow

1. Name the pattern and the communicative job it does.
2. Give the smallest rule that helps the learner act correctly.
3. Contrast it with the learner's likely wrong pattern.
4. Run 3-7 drills: recognition, transformation, cloze, production, mixed review.
5. Record errors with a stable `pattern_id`.

## Feedback Format

For each answer:

- Verdict and corrected version.
- One reason, phrased at the learner's level.
- A minimal pair or contrast when useful.
- Next micro-drill.

## Rules

- Use the learner's native language for explanation unless immersion mode is active.
- Use target language examples that match the learner's level plus one step.
- For languages with politeness/register systems, mark register explicitly.
- Never invent a universal grammar rule when the language has major dialect or register variation.
