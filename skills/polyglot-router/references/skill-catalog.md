# Skill Catalog

Use this catalog when changing names, prompts, or routing behavior.

| Skill | User-facing label | When to route |
|---|---|---|
| `polyglot-router` | Polyglot Router | The learner wants a session, next step, plan, or general language-learning help. |
| `polyglot-level-check` | Polyglot Level Check | Current level is unknown, stale, or needs evidence. |
| `polyglot-roadmap` | Polyglot Roadmap | Target level, deadline, daily time, or goal changes. |
| `polyglot-vocab` | Polyglot Vocab | Due reviews, new vocabulary, collocations, phrase memory. |
| `polyglot-grammar` | Polyglot Grammar | Repeated grammar/form/syntax/register issues. |
| `polyglot-reading` | Polyglot Reading | Text input, comprehension, source-based learning. |
| `polyglot-conversation` | Polyglot Conversation | Role-play, conversation, fluency, repair strategies. |
| `polyglot-writing` | Polyglot Writing | Written text, email, essay, translation, exam writing. |
| `polyglot-notes` | Polyglot Notes | Durable notes, lesson summaries, mistake pages. |
| `polyglot-test` | Polyglot Test | Checkpoints, weekly tests, placement, mock exams. |
| `polyglot-progress` | Polyglot Progress | Status, trend analysis, plan repair. |
| `polyglot-memory` | Polyglot Memory | Persist validated profile, session, SRS, test, and note updates. |
